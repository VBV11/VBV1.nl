Belangrijke veiligheidsadvies:
  • Plak alleen commando's die je vertrouwt. Voer nooit blind commando's uit die iets van internet downloaden en meteen uitvoeren.
  • Zo kun je veiliger werken: git clone → inspecteer de code → verifieer auteur/commit SHA / check release assets en hashes.
  • Als je wilt, kan ik je voorbeeldcommando helpen omzetten naar een veilige instructie (bijv. clone + inspecteer + run) — maar share het commando zelf alleen hier als je dat wil laten controleren.
